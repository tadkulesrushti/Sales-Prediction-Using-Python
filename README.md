# Sales Prediction Using Python

## Project Overview
This project predicts future sales based on advertising expenditure on TV, Radio, and Newspaper using Machine Learning.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Features
- Data Cleaning
- Data Transformation
- Feature Selection
- Data Visualization
- Linear Regression Model
- Sales Prediction
- Future Sales Prediction
- Model Evaluation

## Dataset
Advertising Dataset

## Model Performance
- R² Score: 0.8994
- Mean Absolute Error (MAE): 1.4608

## Conclusion
The Linear Regression model successfully predicts sales based on advertising budgets. TV advertising has the strongest impact on sales, followed by Radio, while Newspaper has the least impact.
